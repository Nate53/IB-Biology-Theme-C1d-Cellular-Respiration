# IB Biology — Theme C: C1d — Cellular Respiration

An interactive single-page lesson for IB Biology HL students covering Theme C, sub-topic C1d:
**cellular respiration** (glycolysis, the link reaction, the Krebs cycle, electron transport, chemiosmosis, and fermentation).

Built by Laughing Coyote Education.

## What's inside

- **Bilingual** (English / Spanish) — toggle from the sidebar
- **Animated step-throughs** for the three core pathways:
  - Glycolysis: glucose → 2 pyruvate (6-step Biozone-style diagram)
  - Link reaction + Krebs cycle: pyruvate → CO₂ + NADH + FADH₂ + ATP (7-step circular animation)
  - Electron transport chain + chemiosmosis: NADH/FADH₂ → ATP via the inner-membrane complexes and ATP synthase (15-step particle animation)
- **Per-section practice and challenge questions** (multiple choice with feedback, short and extended free response with hints and model answers)
- **Compile My Answers** export — generates a plain-text submission of all the student's free-response answers, ready to paste into Google Classroom
- **Glossary tooltips** on key terms, **drag-match terminology check** (Krebs section), **yeast fermentation lab** simulation, and an **ATP yield calculator**
- **Save/resume** — student progress (step state, answers, language) is saved in `localStorage`

## Files

- `theme-c-c1d-cell-respiration.html` — the entire app, single file (HTML + CSS + JS)
- `illustration-of-mitochondrion-small.jpg` — AI-rendered mitochondrion illustration based on an electron micrograph
- `laughing-coyote-logo.jpg` — brand mark
- `vercel.json` — rewrites the deployed root URL `/` to the HTML file

## Local use

Open `theme-c-c1d-cell-respiration.html` in any modern browser. No build step, no server required.

## Deployment

Deployed to Vercel as a static site. The `vercel.json` rewrite serves the app at the project root.
